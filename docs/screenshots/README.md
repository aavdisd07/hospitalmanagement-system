## How to capture (Windows)
1. Run the app: `mvn -q exec:java -Dexec.mainClass="com.mycompany.hospitalmanagementsystem.HospitalManagementSystem"`
2. Login as `admin` / `admin123`
3. Use **Win+Shift+S** to snip each window. Save as PNG with the names above.
4. Commit the files.
